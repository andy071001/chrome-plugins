凤凰读书插件
==============

各种读书网站，最喜欢的要数凤凰读书，实际使用中，有诸多不便。

鉴于此，开发本插件用以提升自己在凤凰读书时用户体验


主要功能
====
1、图书主页点击链接弹出阅读层，相关样式（依Evernote那种简介的样式）

2、弹出层后翻页功能

3、以上，足矣

Enjoy it myself！
